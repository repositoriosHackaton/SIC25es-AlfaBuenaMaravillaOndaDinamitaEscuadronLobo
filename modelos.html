Modelos Utilizados y Proceso de Entrenamiento

Nuestro sistema utiliza dos enfoques principales para la traducción en tiempo real:

  • Modelo CNN: Se encarga de interpretar la imagen de la mano para identificar letras (fingerspelling). Se ha entrenado utilizando técnicas de clasificación con múltiples capas convolucionales.
  • Modelo LSTM: Se utiliza para analizar secuencias de movimientos y captar la dinámica de los gestos. Este modelo es especialmente útil para interpretar la continuidad y el flujo en el lenguaje de señas.

El proceso incluyó la recolección y preprocesamiento de datos, la extracción de landmarks con MediaPipe y el entrenamiento individual de cada modelo. A continuación, se muestran algunas de las gráficas de desempeño obtenidas durante el entrenamiento:
